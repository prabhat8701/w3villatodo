1. Clone the repository:
   ```bash
    git clone https://github.com/prabhat8701/w3villatodo.git

2. Install dependencies:
   - ```bash

     npm install
   - ```bash

     npm install

3. Run the backend server:
   - ```bash
     cd server
     npm start

4. Run the frontend:
   - ```bash
     cd client
     npm start
